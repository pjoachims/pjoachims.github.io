abseits

Offline-first outdoor maps and route planning for hiking and biking.

  • Downloadable map regions stored on-device — vector tiles, terrain, routing graph (~30MB for an Alps region, cities bigger)
  • Client-side A* routing — elevation-aware, typed arrays, works across regions
  • Map styles: topo, satellite + hillshade, contours, 3D terrain
  • Trail intelligence: SAC grades T1–T6, named routes, ~200 POI types
  • Routing profiles for hiking / biking / gravel; offline fuzzy POI search
  • Custom tile pipeline from OSM/SRTM data

MonkeySplit

Full-stack group expense splitting — web + native, with organic users and zero marketing.

  • Flexible splits: equal, by amount, or percentage — per-payer and per-payee
  • Smart settlements: greedy algorithm for minimum transactions; PayPal one-click settle-up
  • 30+ currencies via daily ECB rates; recurring expenses; interactive charts
  • Offline-first PWA: mutations queue offline, flush on reconnect
  • iOS app shipped (Fastlane pipeline)
  • Cloudflare end-to-end: Worker (Hono) + D1 + KV + Pages, Better Auth

🦜 Pivot Parrot

Hand it a DataFrame, get an interactive pivot table.

  • Every aggregation is a real SQL query — DuckDB WASM in-browser, or remote mode via HTTP
  • SQL CUBE queries with caching: rearranging fields is instant
  • Scales to 100k+ rows; virtual scrolling; custom SQL metrics
  • Monorepo: React library + Streamlit wrapper (pip installable) + Jupyter widget
  • Apache Arrow transfer, automatic schema detection, CSV/JSON in, CSV out
